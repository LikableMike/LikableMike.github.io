Beyond - 3D Java Multiplayer Game

This project was for my Computer Game Architecture class where we learned the structure of game engines and how to modify them to fit our needs. Instead of using professional game engines like Unity our professor had created a very minimal game engine in java which forced us to get our hands dirty and build upon it to add features we wanted. I took this course along side an Advanced Computer Graphics course so I was also able to incorperate what I hade learned in that class to enhance the visuals by modifying the shader pipeline. This project taught me about different camera controllers, scenegraphs, and how to build game worlds with textures and skyboxes among other things. In my game I largely focused on the physics aspect as I found it very interesting but also enjoyed building the NPC's and networking allowing for multiplayer. For more information and a deeper description of the game itself check out my GitHub.
